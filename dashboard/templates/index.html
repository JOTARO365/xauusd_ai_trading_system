เปิด Trailing Stop

⚠️

เปิด NNLB Mode — รับทราบความเสี่ยง

  • ข้าม Money Management gates ทั้งหมด — ไม่มีการกรอง confidence, RR, daily loss
  • Lot size scale ตาม equity tier โดยตรง — อาจขาดทุน เกิน % ที่ยอมรับได้ในโหมดปกติ
  • หาก MIN_LOT เกิน Max Loss Budget ระบบ ยังคงเข้า order ที่ MIN_LOT ต่อไป
  • เหมาะสำหรับทุนน้อยที่ยอมรับความเสี่ยงสูง เท่านั้น

โหมดนี้ออกแบบสำหรับผู้ที่เข้าใจความเสี่ยงอย่างครบถ้วนแล้ว
